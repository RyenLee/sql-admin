## 📊 框架对比

### **1. 后端框架选择**

| 框架 | 性能 | 学习曲线 | 生态 | 最佳场景 |
|------|------|---------|------|---------|
| **Axum** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐⭐ | 🏆 现代异步API，易于集成 |
| **Actix Web** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 高负载服务，Actor模型 |
| **Rocket** | ⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐⭐ | MVP/原型，开发者体验优先 |
| **Warp** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ | 函数式编程，WebSocket应用 |

**推荐：Axum** ✅
- 基于Tokio + Tower生态
- 类型安全的request提取
- 最具现代性和灵活性
- 社区增长最快

### **2. 前端框架选择**

**推荐：Leptos** ✅
- 细粒度响应性（类似 SolidJS）

```rust
// packages/shared/src/lib.rs
use serde::{Deserialize, Serialize};
use chrono::{DateTime, Utc};
use uuid::Uuid;

/// 待办事项模型
#[derive(Clone, Debug, Serialize, Deserialize, PartialEq)]
pub struct TodoItem {
    pub id: Uuid,
    pub title: String,
    pub description: Option<String>,
    pub completed: bool,
    pub created_at: DateTime<Utc>,
    pub updated_at: DateTime<Utc>,
}

/// 创建待办事项请求
#[derive(Clone, Debug, Deserialize)]
pub struct CreateTodoRequest {
    pub title: String,
    pub description: Option<String>,
}

/// 更新待办事项请求
#[derive(Clone, Debug, Deserialize)]
pub struct UpdateTodoRequest {
    pub title: Option<String>,
    pub description: Option<String>,
    pub completed: Option<bool>,
}

/// API响应包装器
#[derive(Serialize)]
pub struct ApiResponse<T> {
    pub success: bool,
    pub data: Option<T>,
    pub error: Option<String>,
}

impl<T> ApiResponse<T> {
    pub fn ok(data: T) -> Self {
        Self {
            success: true,
            data: Some(data),
            error: None,
        }
    }

    pub fn err(error: impl Into<String>) -> Self {
        Self {
            success: false,
            data: None,
            error: Some(error.into()),
        }
    }
}
```

### **3. 后端：Axum主应用**

```rust
// packages/backend/src/main.rs
mod handlers;
mod models;
mod db;
mod middleware;
mod error;
mod state;
mod config;

use axum::{
    Router,
    routing::{get, post, put, delete},
    middleware as axum_middleware,
};
use std::sync::Arc;
use sqlx::PgPool;
use tower_http::{
    trace::TraceLayer,
    cors::CorsLayer,
};
use tracing_subscriber;

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    // 1. 初始化日志
    tracing_subscriber::fmt()
        .with_env_filter(
            tracing_subscriber::EnvFilter::from_default_env()
                .add_directive("rust_fullstack=debug".parse()?),
        )
        .init();

    // 2. 加载环境变量
    dotenv::dotenv().ok();
    let config = config::AppConfig::from_env()?;

    // 3. 初始化数据库连接池
    let pool = PgPool::connect(&config.database_url).await?;
    sqlx::migrate!("./migrations")
        .run(&pool)
        .await?;

    // 4. 创建应用状态
    let app_state = Arc::new(state::AppState {
        db_pool: pool,
        config: config.clone(),
    });

    // 5. 构建路由
    let router = Router::new()
        // 待办事项路由
        .route("/api/todos", 
            post(handlers::todos::create_todo)
                .get(handlers::todos::list_todos)
        )
        .route("/api/todos/:id",
            get(handlers::todos::get_todo)
                .put(handlers::todos::update_todo)
                .delete(handlers::todos::delete_todo)
        )
        // 健康检查
        .route("/health", get(health_check))
        // 中间件
        .layer(axum_middleware::from_fn(middleware::logging))
        .layer(TraceLayer::new_for_http())
        .layer(CorsLayer::permissive())
        // 状态注入
        .with_state(app_state);

    // 6. 启动服务器
    let listener = tokio::net::TcpListener::bind(&config.server_addr).await?;
    tracing::info!("Server running on {}", config.server_addr);
    
    axum::serve(listener, router).await?;

    Ok(())
}

async fn health_check() -> &'static str {
    "OK"
}
```

### **4. 后端：处理器 (Handlers)**

```rust
// packages/backend/src/handlers/todos.rs
use axum::{
    extract::{State, Path},
    http::StatusCode,
    Json,
};
use uuid::Uuid;
use std::sync::Arc;
use crate::state::AppState;
use shared::{TodoItem, CreateTodoRequest, UpdateTodoRequest, ApiResponse};

/// 创建待办事项
pub async fn create_todo(
    State(state): State<Arc<AppState>>,
    Json(req): Json<CreateTodoRequest>,
) -> (StatusCode, Json<ApiResponse<TodoItem>>) {
    let id = Uuid::new_v4();
    let now = chrono::Utc::now();

    let result = sqlx::query_as!(
        TodoItem,
        r#"
        INSERT INTO todos (id, title, description, completed, created_at, updated_at)
        VALUES ($1, $2, $3, false, $4, $5)
        RETURNING id, title, description, completed, created_at, updated_at
        "#,
        id,
        req.title,
        req.description,
        now,
        now,
    )
    .fetch_one(&state.db_pool)
    .await;

    match result {
        Ok(todo) => (StatusCode::CREATED, Json(ApiResponse::ok(todo))),
        Err(e) => {
            tracing::error!("Failed to create todo: {}", e);
            (
                StatusCode::INTERNAL_SERVER_ERROR,
                Json(ApiResponse::err("Failed to create todo")),
            )
        }
    }
}

/// 获取所有待办事项
pub async fn list_todos(
    State(state): State<Arc<AppState>>,
) -> Json<ApiResponse<Vec<TodoItem>>> {
    let result = sqlx::query_as!(
        TodoItem,
        "SELECT id, title, description, completed, created_at, updated_at FROM todos ORDER BY created_at DESC"
    )
    .fetch_all(&state.db_pool)
    .await;

    match result {
        Ok(todos) => Json(ApiResponse::ok(todos)),
        Err(e) => {
            tracing::error!("Failed to fetch todos: {}", e);
            Json(ApiResponse::err("Failed to fetch todos"))
        }
    }
}

/// 获取单个待办事项
pub async fn get_todo(
    State(state): State<Arc<AppState>>,
    Path(id): Path<Uuid>,
) -> Result<Json<TodoItem>, StatusCode> {
    sqlx::query_as!(
        TodoItem,
        "SELECT id, title, description, completed, created_at, updated_at FROM todos WHERE id = $1",
        id
    )
    .fetch_optional(&state.db_pool)
    .await
    .map_err(|_| StatusCode::INTERNAL_SERVER_ERROR)?
    .ok_or(StatusCode::NOT_FOUND)
    .map(Json)
}

/// 更新待办事项
pub async fn update_todo(
    State(state): State<Arc<AppState>>,
    Path(id): Path<Uuid>,
    Json(req): Json<UpdateTodoRequest>,
) -> Result<Json<TodoItem>, StatusCode> {
    let now = chrono::Utc::now();

    sqlx::query_as!(
        TodoItem,
        r#"
        UPDATE todos
        SET
            title = COALESCE($2, title),
            description = COALESCE($3, description),
            completed = COALESCE($4, completed),
            updated_at = $5
        WHERE id = $1
        RETURNING id, title, description, completed, created_at, updated_at
        "#,
        id,
        req.title,
        req.description,
        req.completed,
        now,
    )
    .fetch_optional(&state.db_pool)
    .await
    .map_err(|_| StatusCode::INTERNAL_SERVER_ERROR)?
    .ok_or(StatusCode::NOT_FOUND)
    .map(Json)
}

/// 删除待办事项
pub async fn delete_todo(
    State(state): State<Arc<AppState>>,
    Path(id): Path<Uuid>,
) -> StatusCode {
    match sqlx::query!("DELETE FROM todos WHERE id = $1", id)
        .execute(&state.db_pool)
        .await
    {
        Ok(result) if result.rows_affected() > 0 => StatusCode::NO_CONTENT,
        _ => StatusCode::NOT_FOUND,
    }
}
```

```rust
// packages/frontend/src/api/client.rs
use leptos::prelude::*;
use shared::{TodoItem, CreateTodoRequest, UpdateTodoRequest, ApiResponse};

const API_BASE: &str = "http://localhost:3000/api";

pub async fn fetch_todos() -> Result<Vec<TodoItem>, String> {
    let response = gloo_net::http::Request::get(&format!("{}/todos", API_BASE))
        .send()
        .await
        .map_err(|e| e.to_string())?;

    let api_response: ApiResponse<Vec<TodoItem>> = response
        .json()
        .await
        .map_err(|e| e.to_string())?;

    api_response
        .data
        .ok_or_else(|| api_response.error.unwrap_or_default())
}

pub async fn create_todo(req: CreateTodoRequest) -> Result<TodoItem, String> {
    let response = gloo_net::http::Request::post(&format!("{}/todos", API_BASE))
        .json(&req)
        .map_err(|e| e.to_string())?
        .send()
        .await
        .map_err(|e| e.to_string())?;

    let api_response: ApiResponse<TodoItem> = response
        .json()
        .await
        .map_err(|e| e.to_string())?;

    api_response
        .data
        .ok_or_else(|| api_response.error.unwrap_or_default())
}

pub async fn update_todo(
    id: uuid::Uuid,
    req: UpdateTodoRequest,
) -> Result<TodoItem, String> {
    let response = gloo_net::http::Request::put(&format!("{}/todos/{}", API_BASE, id))
        .json(&req)
        .map_err(|e| e.to_string())?
        .send()
        .await
        .map_err(|e| e.to_string())?;

    let api_response: ApiResponse<TodoItem> = response
        .json()
        .await
        .map_err(|e| e.to_string())?;

    api_response
        .data
        .ok_or_else(|| api_response.error.unwrap_or_default())
}

pub async fn delete_todo(id: uuid::Uuid) -> Result<(), String> {
    gloo_net::http::Request::delete(&format!("{}/todos/{}", API_BASE, id))

        .await
```

### **7. 前端：待办页面**

```rust
// packages/frontend/src/pages/todos.rs
use leptos::prelude::*;
use leptos_router::hooks::use_navigate;
use crate::api::client;
use shared::{TodoItem, CreateTodoRequest};

#[component]
pub fn TodosPage() -> impl IntoView {
    let navigate = use_navigate();
    
    // 异步资源：获取待办事项列表
    let todos_resource = Resource::new(
        || (),
        |_| async {
            client::fetch_todos().await.unwrap_or_default()
        }
    );

    // 新增输入框状态
    let (title, set_title) = create_signal(String::new());

    // 处理创建待办事项
    let handle_create = move |_| {
        let title_val = title.get();
        if title_val.is_empty() {
            return;
        }

        spawn_local(async move {
            match client::create_todo(CreateTodoRequest {
                title: title_val.clone(),
                description: None,
            }).await {
                Ok(_) => {
                    set_title(String::new());
                    todos_resource.refetch();
                }
                Err(e) => {
                    tracing::error!("Failed to create todo: {}", e);
                }
            }
        });
    };

    view! {
        <div class="max-w-2xl mx-auto py-8 px-4">
            <h1 class="text-3xl font-bold mb-6">"My Todos"</h1>

            // 创建新待办
            <div class="mb-6 flex gap-2">
                <input
                    type="text"
                    class="flex-1 px-4 py-2 border rounded"
                    placeholder="Add a new todo..."
                    value=title
                    on:input=move |ev| set_title(event_target_value(&ev))
                    on:keydown=move |ev| {
                        if ev.key() == "Enter" {
                            handle_create(());
                        }
                    }
                />
                <button
                    class="px-6 py-2 bg-blue-600 text-white rounded hover:bg-blue-700"
                    on:click=handle_create
                >
                    "Add"
                </button>
            </div>

            // 待办列表
            <Suspense fallback=move || view! { <p>"Loading..."</p> }>
                {move || {
                    todos_resource.and_then(|todos| {
                        if todos.is_empty() {
                            return view! {
                                <p class="text-gray-500">"No todos yet. Create one to get started!"</p>
                            }.into_any();
                        }

                        view! {
                            <ul class="space-y-2">
                                {todos.into_iter().map(|todo| {
                                    view! {
                                        <TodoItemComponent todo=todo on_change=move |_| {
                                            todos_resource.refetch();
                                        } />
                                    }
                                }).collect_view()}
                            </ul>
                        }.into_any()
                    })
                }}
            </Suspense>
        </div>
    }
}

#[component]
fn TodoItemComponent(
    todo: TodoItem,
    on_change: impl Fn(()) + 'static,
) -> impl IntoView {
    let (completed, set_completed) = create_signal(todo.completed);

    let handle_toggle = move |_| {
        let new_completed = !completed.get();
        set_completed(new_completed);
        
        let todo_id = todo.id;
        spawn_local(async move {
            let _ = client::update_todo(
                todo_id,
                shared::UpdateTodoRequest {
                    title: None,
                    description: None,
                    completed: Some(new_completed),
                }
            ).await;
        });
    };

    let handle_delete = move |_| {
        let todo_id = todo.id;
        spawn_local(async move {
            if let Ok(_) = client::delete_todo(todo_id).await {
                on_change(());
            }
        });
    };

    view! {
        <li class="flex items-center gap-3 p-3 bg-white rounded shadow">
            <input
                type="checkbox"
                checked=completed
                on:change=handle_toggle
                class="w-5 h-5"
            />
            <span class=move || if completed.get() {
                "line-through text-gray-500 flex-1"
            } else {
                "flex-1"
            }>
                {todo.title.clone()}
            </span>
            <button
                class="px-3 py-1 text-red-600 hover:bg-red-50 rounded"
                on:click=handle_delete
            >
                "Delete"
            </button>
        </li>
    }
}
```
